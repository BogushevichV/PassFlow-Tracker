В авалонии мы будем работать в вижуал студио. Далее инструкция по установке авалонии.
Для создания проекта на Avalonia UI через Visual Studio вам нужно скачать одну важную вещь, так как по умолчанию шаблонов Avalonia в Visual Studio нет.

Что нужно сделать:
1. Установить расширение (Обязательно)
Чтобы в меню «Создать проект» появились шаблоны Avalonia и работал визуальный предпросмотр интерфейса (Designer), нужно установить расширение:

Откройте Visual Studio.

Перейдите в меню Extensions (Расширения) → Manage Extensions (Управление расширениями).

В поиске введите «Avalonia».

Найдите Avalonia for Visual Studio 2022 и нажмите Download.

После этого закройте Visual Studio, чтобы запустилась установка.

2. Установить шаблоны через консоль (Рекомендуется)
Хотя расширение добавит шаблоны, иногда полезно установить их и в саму систему .NET, чтобы они всегда были доступны через командную строку. Для этого откройте терминал (PowerShell или CMD) и введите:

Bash

dotnet new install Avalonia.Templates
Как создать проект после установки:
Запустите Visual Studio.

Нажмите Create a new project (Создание проекта).

В поле поиска введите Avalonia.

Выберите подходящий шаблон (например, Avalonia .NET MVVM App — это стандарт для серьезных приложений).

Нажмите «Далее» и следуйте инструкциям.

Важный нюанс:
Убедитесь, что у вас установлена рабочая нагрузка «.NET desktop development» (Разработка классических приложений .NET) в Visual Studio Installer. Без неё Avalonia работать не будет, так как она опирается на базу .NET.


После установки просто клонируй гит репозиторий.
